Десктопное приложение на C++ и Qt для моделирования сети аэропортов и поиска оптимальных авиамаршрутов. Проект сочетает алгоритмическую базу на графах и графический интерфейс для управления полетами.

## Стек технологий
* Язык C++.
* Фреймворк Qt (Widgets)
* Формат данных JSON для импорта и экспорта конфигурации сети.

## Алгоритмическая база
* **Алгоритм Дейкстры:** Поиск кратчайшего пути по критериям: цена, время и комбинированный вес
* **Алгоритм Йена (Yen's algorithm):** Нахождение K-кратчайших путей (альтернативных маршрутов без циклов)
* **Графовые фильтры:** Кастомная логика для динамического исключения узлов и ребер при поиске альтернатив
* **Валидация стыковок:** Учет минимального времени пересадки между рейсами

## Архитектура
Проект реализован с четким разделением логики и интерфейса:
1. **Core Logic (PathFinder):** Движок расчетов. Граф представлен как список смежности на базе `std::unordered_map`.
2. **Data Models (Airport, Flight):** Структуры данных для описания компонентов сети.
3. **UI Layer (Dialog):** Слой интерфейса на Qt для редактирования

## Сборка
Для сборки требуется:
1. Qt 5.15+ или Qt 6.
2. Компилятор с поддержкой C++17 (MSVC, GCC или Clang).
3. Открыть файл проекта (CMakeLists.txt) в Qt Creator и запустить сборку.
